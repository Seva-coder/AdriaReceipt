Приложение предназначено для ведения статистики расходов и представления её в наглядном виде.
Для этого есть несколько функций.
----------
Для начала добавим чек. Открывается сканер qr-кодов. Сканируем код, немного ждём. Бывает что Черногорское апи сильно тормозит, и ждать приходится почти минуту.

Итак открывается чек. Переименуем магазин для удобного отображения в списках. Для каждого товара указываем название, группу товара (создадим её прямо здесь). Указываем единицу измерения - она пригодится для отображения.
Заполнять заново не придётся - если товар был добавлен, эти поля будут предзаполнены.

Сохраняем чек.
Если нет бумажного чека нет, можно заполнить всё вручную из этого меню. Ну, здесь всё аналогично, кроме сканирования.
----------
раздел Расходы
При достаточном количестве чеков, можно посмотреть расход в табличном виде за разные промежутки времени - например за 7 дней, 30, или выбрать необходимый период. Группы товаров, по которым проводится расчёт расходов можно указать галочками сверху, например, оставим только сладкое, или мясо. Таблица при этом обновляется. Опять же повторю, что сами группы создаёт пользователь. По умолчанию есть только группа Products.
----------
В разделе товары можно посмотреть, сколько конкретный товар стоил в разных магазинах. Сравниваются последние цены по каждому магазину. Самый дешёвый вариант подсвечивается зелёным. Так же отображаются исходные названия из чеков, его группа и траты на этот товар.
Ещё здесь можно отредактировать - изменить название, группу, единицу измерения.
----------
Раздел магазины/группы нужен для переименования соответствующих магазинов или групп. Возможно в следующих версиях добавлю сюда ещё какой-нибудь функционал.
----------
В разделе чеки можно посмотреть интересующий чек. Если в нём были одинаковые товары, то они объединятся в один, а их количество и стоимость просуммируются соответственно.
----------
Раздел графики. Наверно самая наглядная часть представлений. В левой вкладке находятся диаграммы, показывающие соотношение расходов по группам за интересующий период. Отобразятся только те группы, товары из которых были куплены в эти периоды.
В правой вкладке отображение расходов по времени.
----------
Вот, пожалуй, и всё. В дальнейшем возможно добавлю возможность сканирования в других странах, а так же новые функции.
