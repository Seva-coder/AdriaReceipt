name:
AdriaReceipt


short:
Ведение статистики расходов с помощью сканирования чеков в Черногории.



long:
Приложение предназначено для ведения статистики расходов с помощью сканирования QR-кодов магазинных чеков. Это позволяет сэкономить, например, покупая в магазинах где товар стоит дешевле. Сканировать QR-код можно также сторонним приложением (например, Binary Eye).
После сканирования чека загружается его электронная версия. Поддерживаемые языки: русский, сербский. Реклама полностью отсутствует. Есть поддержка тёмной темы.

В приложении есть несколько функций, позволяющих понять расходы:

1. Список с расходами по каждому товару за определённый период времени. Можно указать какие группы учитывать для отображения и расчёта.
2. Сравнение стоимости товара в разных магазинах, подсвечивает минимальную цену. Для сравнения берутся последние цены по каждому магазину.
3. Круговые диаграммы для наглядного отображения расходов по группам за период времени. При большом количестве групп экран можно двигать вверх-вниз.
4. Диаграммы расходов по времени. Каждый столбик - небольшой период времени (день/неделя/месяц), его высота зависит расходов в этот период.
5. Список с чеками - можно посмотреть сохранённые чеки.
6. Так же есть вспомогательные функции редактирования товаров, групп, магазинов.

При добавлении чека необходимо для каждого нового товара указать его имя - для более понятного отображения, выбрать группу или добавить новую - для возможности оценки расходов по группам, единицу измерения. Уже добавленные товары будут подсвечены зелёным. Если товарам с различными названиями в чеках (например, "картофель" и "картофель молодой") указать одно имя - то в таблице расходов они будут объединены и отобразятся одной строкой.

В данный момент приложение поддерживает только чеки из магазинов Черногории, планируется добавление и других стран.
Про найденные баги и ошибки перевода пишите на e-mail (указан в google play), или в telegram-чат по ссылке из справки в приложении.
